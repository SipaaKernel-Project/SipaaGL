# SipaaGL
The Sipaa Graphics Library (Forked from [PrismGraphics](https://github.com/Project-Prism/Prism-OS/tree/main/PrismGraphics))

## Features
SVGA / VBE Canvas
BitFont, PSF, TTF String Drawing
Custom Color class
Animations
Gaussian Blur (in dev)
Gradients (broken)
3D Rasterizer
